Cara Pakai Singkat (Offline):
1. Ekstrak ZIP ke komputer/HP.
2. Buka file Smart_Steps_TKA_Session1.html di browser (double click atau 'Open with' → Chrome).
3. Siswa bisa mengerjakan kuis tanpa koneksi internet. Skor akan ditampilkan di akhir.

Menambah Session Baru (Session 2–4):
1. Duplikat file Smart_Steps_TKA_Session1.html → beri nama Smart_Steps_TKA_Session2.html.
2. Buka file dengan editor teks (Notepad / VSCode). Cari bagian 'const questions = [' dan ganti isi soal sesuai sesi baru.
3. Simpan, lalu uji buka file di browser. Jika ingin dipublikasi, upload file baru ke Netlify/GitHub sama seperti langkah di panduan.
